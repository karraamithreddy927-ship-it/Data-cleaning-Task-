## Analysis
- Customers with fewer login days are more likely to cancel.
- Customers with more support calls show higher churn rates.
- Longer subscription periods indicate better customer retention.

## Conclusion
Customer churn is strongly related to user activity, support interactions, and subscription duration. Improving engagement and customer support can help reduce churn.
